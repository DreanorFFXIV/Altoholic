# Altoholic
Everything you need for your alts.

Features
* Relog to your alt characters (Requires [Autoretainer](https://puni.sh/plugin/AutoRetainer))
* Currencies for all characters
  * Common
  * Battle
  * Other
  * Tribal
* Collections for all characters
  * Mounts
  * Minions
  * Emotes
  * Triple Triad Cards
  * Orchestrion Rolls
